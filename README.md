# coffe-shop

vue + vant 

## 技术栈
- Vue 
- Vue Router
- Vuex 
- Vue CLI
- Vant
- Sass 
- ES6
- Axios


## 安装与使用
```shell
# 安装依赖
npm install

# 本地运行
npm run serve 	

# 打包
npm run build		
```

## 功能
1.  登录注册
2.  主页
3.  菜单
4.  商品详细页
5.  购物袋
6.  我的
7.  结算
8.  我的订单
9.  收货地址
10. 我的收藏
11. 个人信息
12. 账号与安全
